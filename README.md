A simple multi-algorithm proof-of-work library for Rust.

## Algorithms

- [x] SHA-256
- [x] SHA-512
- [x] Scrypt
- [ ] Argon2id

API references are available at [docs.rs/rspow](https://docs.rs/rspow).
